Medi-scan-Image-analysis-for-Diabetic-Retinopathy

This project focuses on developing a Python-based system for detecting Diabetic Retinopathy (DR) using deep learning. DR is a diabetes-related eye disease that can lead to blindness if not diagnosed early. The proposed model uses Convolutional Neural Networks (CNNs) trained on retinal fundus images to classify the severity of DR into five stages.
