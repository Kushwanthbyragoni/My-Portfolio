# 🚀 Kushwanth Byragoni — Personal Portfolio

> A futuristic, premium personal portfolio website built with pure HTML, CSS, and JavaScript. Designed with a Tech + Automotive + Anime fusion aesthetic.


---

## ✨ Features

- **Animated Loading Screen** — Progress bar loader with smooth fade-out
- **Custom Particle Canvas** — 80 floating particles with dynamic connection lines
- **Scroll Reveal Animations** — Elements fade in as you scroll
- **Animated Skill Bars** — Progress bars animate on scroll into view
- **Futuristic UI** — Glassmorphism cards, glowing cyan/orange borders, racing line effects
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Contact Form** — Ready to wire up to a backend or EmailJS
- **Smooth Navigation** — Fixed navbar with active link underlines

## 🚀 Getting Started

### Option 1 — Open directly
Just open `index.html` in your browser. No server needed.

### Option 2 — Live Server (recommended)
If you use VS Code:
1. Install the **Live Server** extension
2. Right-click `index.html` → **Open with Live Server**

---



### Change colors
In `style.css`, edit the `:root` variables:

```css
:root {
    --cyan:   #00f5ff;   /* Primary accent */
    --orange: #ff6b00;   /* Secondary accent */
    --gold:   #ffd700;   /* Luxury accent */
    --pink:   #ff2d78;   /* Anime accent */
    --bg:     #020510;   /* Background */
}
```

## 📦 Deployment



### Deploy on GitHub Pages (Free)
1. Push your code to GitHub
2. Go to **Settings** → **Pages**
3. Set source to `main` branch → `/root`
4. Your site will be live at `https://yourusername.github.io/repo-name`
